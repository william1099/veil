Cara Menggunakan Aplikasi

Veil adalah aplikasi mobile yang digunakan untuk melakukan lelang secara online dengan mudah dan aman. Aplikasi ini menggunakan virtual coin sebagai alat untuk melakukan pertukaran. Virtual Coin (VC) mempunyai prinsip yang sama dengan bitcoin yakni dapat dibeli dan dapat ditukarkan dengan uang.


Tata cara penggunaan aplikasi

  1. User melakukan registrasi dengan data valid
  2. User mengisi data diri untuk keperluan transaksi
  3. User memilih barang yang ingin diBid
  4. Barang yang sudah diBid tidak bisa di cancel
  5. VC akan terpotong otomatis ketika barang berhasil menjadi milik user
  6. Setelah barang sampai pada user, VC akan langsung dikirimkan ke pelelang


Mengapa menggunakan virtual coin ?

Agar transaksi dapat dilakukan dengan aman dan nyaman

Bagaimana caranya membeli VC ?

Untuk membeli VC(Virtual Coin) ,bisa langsung menghubungi email veilindonesia@gmail.com
